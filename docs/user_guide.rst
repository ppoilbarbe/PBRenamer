User Guide
==========

Getting Started
---------------

Launch PBRenamer from the command line::

    pbrenamer

Or via the application menu if installed via a desktop package.

Usage
-----

1. Click **Open folder…** to select the directory containing files to rename.
2. Configure the renaming rules in the panel on the right.
3. Preview the result in the file list.
4. Click **Rename** to apply.
